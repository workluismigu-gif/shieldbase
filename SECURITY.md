# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

**Email:** security@shieldbase.io

We acknowledge receipt within 48 hours and provide a fix timeline within 7 days.

## Security Measures

- Data encrypted at rest and in transit (AES-256 / TLS 1.2+)
- SOC 2 Type I compliance in progress
- Automated security scanning via Prowler
- Dependency vulnerability scanning via Dependabot
- Secret scanning enabled on all repositories
